# 🛡️ Ciberseguridad: Laboratorios Prácticos (Google Certificate)

¡Bienvenido a mi repositorio de aprendizaje! Aquí documento mis habilidades técnicas desarrolladas durante el Certificado Profesional de Ciberseguridad de Google.

## 🚀 Proyectos y Habilidades
- **Linux:** Gestión de permisos y análisis de logs mediante línea de comandos (CLI).
- **SQL:** Filtrado de bases de datos para identificar brechas de seguridad.
- **Python:** Automatización de tareas repetitivas de seguridad.
- **Redes:** Análisis de tráfico con Wireshark y Packet Sniffers.

## 📂 Estructura del Repositorio
- `Linux-Labs/`: Comandos de administración y seguridad.
- `SQL-Queries/`: Consultas de auditoría y filtrado.
- `Assets/`: Capturas de pantalla de los laboratorios.
